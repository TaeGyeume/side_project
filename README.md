@latest 뒤에붙히면 안정화버전설치

###### server

MongoDB와 상호작용하기 위한 라이브러리
npm install mongoose

# passport-facebook
npm install passport-facebook

# bcrypt (혹은 대체 패키지 bcryptjs)
npm install bcrypt
# 또는
npm install bcryptjs

# multer
npm install multer

###### client

Bootstrap 5 아이콘 사용
npm install bootstrap-icons
